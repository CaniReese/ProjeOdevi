# Fitness Merkezi Yönetim Sistemi

Bu proje, ASP.NET Core MVC mimarisi kullanılarak geliştirilmiş bir fitness merkezi yönetim sistemidir. 
Sistem, kullanıcı yönetimi, randevu planlama ve yapay zekâ destekli egzersiz planı oluşturma gibi 
modern web uygulamalarında sıkça kullanılan özellikleri bir araya getirmektedir.

## Kullanılan Teknolojiler
- ASP.NET Core MVC
- Entity Framework Core
- Microsoft SQL Server
- ASP.NET Core Identity
- OpenAI API
- Bootstrap

## Temel Özellikler
- Kullanıcı kayıt ve giriş işlemleri
- Rol bazlı yetkilendirme (Admin / Kullanıcı)
- Eğitmen ve hizmet türü yönetimi
- Randevu oluşturma ve takip sistemi
- Eğitmen müsaitlik takibi
- Yapay zekâ destekli kişisel egzersiz planı oluşturma

## Yapay Zekâ Entegrasyonu
Projede OpenAI API kullanılarak bir **AI Koç** modülü geliştirilmiştir. 
Bu modül sayesinde kullanıcılar yaş, boy, kilo ve hedef bilgilerini girerek 
kendilerine özel 7 günlük egzersiz planları oluşturabilmektedir.

Yapay zekâ entegrasyonu, kullanıcı deneyimini artırmakta ve sistemi klasik bir 
yönetim uygulamasının ötesine taşımaktadır.

## Proje Amacı
Bu projenin amacı, gerçek hayatta kullanılabilecek bir fitness merkezi yönetim sistemi 
geliştirerek ASP.NET Core MVC mimarisinin, veritabanı ilişkilerinin ve yapay zekâ 
entegrasyonunun birlikte nasıl kullanılabileceğini göstermektir.
