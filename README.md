# Fitness Center Management System

This project is an ASP.NET Core MVC application developed for a university course.

## Features
- User authentication with ASP.NET Identity
- Trainer and service management
- Appointment scheduling
- Role-based authorization (Admin, Trainer, User)

## Technologies
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- SQL Server
- Bootstrap
